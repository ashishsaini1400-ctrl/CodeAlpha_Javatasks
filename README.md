# Java Projects

This repository contains Java-based projects developed using Object-Oriented Programming concepts.

## Project 1: Student Grade Tracker

### Description
A console-based Student Grade Tracker application developed in Java.
This program allows users to input, manage, and analyze student grades.

### Features
- Input and store student grades
- Manage student records using ArrayList
- Calculate average score of students
- Find highest and lowest scores
- Display summary report of all students
- Console-based user interface

### Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList

## Project 2: Stock Trading Platform

### Description
A basic Stock Trading Platform developed using Java that simulates a stock trading environment.
Users can view market data, perform buy and sell operations, and manage their investment portfolio.

### Features
- Display available market stock data
- Buy stocks with quantity management
- Sell stocks from portfolio
- Track portfolio performance
- Maintain transaction history
- Manage stocks, users, and transactions using OOP concepts
- Save portfolio data using File I/O

### Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- File Handling
